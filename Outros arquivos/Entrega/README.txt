Os parâmetros recebidos por simulatedAnnealing.py são descritos ao chamá-lo com o argumento -h.

Ressalta-se que os dois primeiros são obrigatórios e posicionais. O primeiro deve ser o caminho para um arquivo em que o log da execução do algoritmo será salvo; o segundo deve ser o caminho para um arquivo que contenha uma instância do GBMV. Recomenda-se botar o caminho sempre entre aspas, como no exemplo abaixo.

Exemplo de caminho para a criação de um arquivo de log:
"D:/UFRGS/Quinto Semestre/Otimização Combinatória/Trabalho final/Outputs/log.txt"
